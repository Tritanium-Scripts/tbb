Tritanium Bulletin Board 2.0 - Alpha 2



Allgemeine Hinweise:

Bei dieser Version handelt es sich um eine Alphaversion!! Das bedeutet, es handelt sich um eine noch sehr sehr frühe Version, bei der sich im Extremfall noch komplett alles ändern kann! Des Weiteren sind vorhandene Fehler nicht auszuschließen, eher sogar sehr wahrscheinlich! Ich empfehle außerdem nicht, diese Version für den Einsatz in einem längeren Zeitraum einzuplanen, da es für spätere Alpha- und Betaversionen keine Updates geben wird!
Diese Version wurde in erster Linie zum testen freigegeben, es werden für die Installation allgemein erweiterte Kenntnisse empfohlen.
Über Hinweise, Verbesserungsvorschläge und Anregungen vor allem die Datenstruktur und deren Aufbau betreffend würde ich mich sehr freuen!


Installation

- Alles aus dem Archiv uploaden, die Ordnerstruktur übernehmen (die Ordner "docs" und "sdk" können aber weggelassen werden). Dabei beachten, dass alle .php Dateien im Ascii-Modus, alle restlichen Dateien im Binärmodus upgeloaded werden.
- Folgende Rechte verteilen:
    .\*.php 755
    .\data 755
    .\data\forums 777
    .\data\topics 777
    .\data\users 777
    .\data\vars 755
    .\data\vars\*.dat 777
    .\db 755
    .\db\filedb 755
    .\db\filedb\*.php 755
    .\language 755
    .\language\german 755
    .\language\german\*.php 755
    .\templates 755
    .\templates\standard 755
    .\templates\standard\*.* 644
    .\templates\standard\*.php 755
    .\templates\standard\images 755
    .\templates\standard\images\*.* 644
- Das Forum per index.php aufrufen und ein bisschen rumprobieren...
- Fehler, Verbesserungsvorschläge, Anregungen usw. bitte bitte per Mail an julian@tritanium-scripts.de
