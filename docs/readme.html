Allgemeine Hinweise

Bei dieser Version handelt es sich um eine Alphaversion!! Das bedeutet, es handelt sich um eine noch sehr sehr frühe Version, bei der sich im Extremfall noch komplett alles ändern kann! Des Weiteren sind vorhandene Fehler nicht auszuschlieﬂen, eher sogar sehr wahrscheinlich! Ich empfehle auﬂerdem nicht, diese Version für den Einsatz in einem längeren Zeitraum einzuplanen, da es für spätere Alpha- und Betaversionen keine Updates geben wird!
Diese Version wurde in erster Linie zum testen freigegeben, es werden für die Installation allgemein erweiterte Kenntnisse empfohlen.
Über Hinweise, Verbesserungsvorschläge und Anregungen vor allem die Datenstruktur und deren Aufbau betreffend würde ich mich sehr freuen!



Installation

- Alles aus dem Archiv uploaden, die Ordnerstruktur übernehmen (die Ordner "docs" und "misc" sollten aber weggelassen werden). Dabei beachten, dass alle .php Dateien im Ascii-Modus, alle restlichen Dateien im Binärmodus upgeloaded werden.
- Folgende Rechte verteilen:
    .\*.php 755
    .\cache 777
    .\dbconfig.php 777 (nach der Installation wieder ändern!)
    .\addons 755
    .\images 755
    .\images\postpics 755
    .\images\postpics\*.* 644
    .\images\rankpics 755
    .\images\rankpics\*.* 644
    .\images\smilies 755
    .\images\smilies\*.* 644
    .\db 755
    .\db\*.php 755
    .\db\*.sql 644
    .\language 755
    .\language\german 755
    .\language\german\*.php 755
    .\language\german\emails 755
    .\language\german\emails\*.tpl 644
    .\templates 755
    .\templates\ts_tbb2_standard 755
    .\templates\ts_tbb2_standard\cache 777
    .\templates\ts_tbb2_standard\*.* 644
    .\templates\ts_tbb2_standard\*.php 755
    .\templates\ts_tbb2_standard\images 755
    .\templates\ts_tbb2_standard\images\*.* 644
    .\upload 755
    .\upload\avatars 777
    .\upload\files 777
- Setzen Sie die Rechte der Datei "dbconfig.php" auf 777 (nach der Installation wieder ändern!)
- Die Installation per install.php aufrufen und den Anweisungen folgen
- Nach der Installation die Rechte der Datei "dbconfig.php" auf 755 setzen und "install.php" löschen
- Das Forum per index.php aufrufen und ein bisschen rumprobieren...
- Fehler, Verbesserungsvorschläge, Anregungen usw. bitte per Mail an julian@tritanium-scripts.com oder ins Forum unter http://www.tritanium-scripts.com
