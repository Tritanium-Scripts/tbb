Allgemeine Hinweise

Bei dieser Version handelt es sich um eine Betaversion! Dies bedeutet, dass der Großteil der Funktionalität weitgehend fehlerfrei zur Verfügung steht, die Software aber weder auf Sicherheit noch auf Geschwindigkeit überprüft und/oder optimiert wurde. Von einem Einsatz in einem produktiven Umfeld wird dringend abgeraten!

Installation

  • Das komplette Archiv (z.B. per FTP) auf den Webserver hochladen, die Ordnerstruktur übernehmen (der Ordner "docs" sollte aber weggelassen werden).
  • Grundsätzlich sollten alle Ordner die Rechte 0700 und alle Dateien die Rechte 0600 besitzen. Ist der Webserver schlecht konfiguriert, kann es helfen die Rechte auf 0755 für Ordner und auf 0644 für Dateien zu setzen. Sollten damit immernoch Probleme bestehen, können noch folgende einzelne Spezialrechte gesetzt werden. Allerdings ist dies sehr sicherheitskritisch und sollte wenn möglich vermieden werden!
    • ./cache 777
    • ./config/DB.config.class.php 777
    • ./uploads/avatars 777
    • ./uploads/files 777
  • Die Installation per install.php aufrufen und den Anweisungen folgen
  • Nach der Installation "install.php" löschen
  • Das Forum per index.php aufrufen
  • Fehler, Verbesserungsvorschläge, Anregungen usw. bitte per Mail an julian@tritanium-scripts.com oder ins Forum unter http://www.tritanium-scripts.com

Update von einer älteren Version des Tritanium Bulletin Board 2

Erstellen Sie am besten einen neuen Ordner für die neue Forenversion und folgenden Sie der Installation wie oben beschrieben. Bei der Suche nach einer vorhandenen Installation sollte die ältere Version automatisch gefunden und geupdatet werden. Daraufhin fahren Sie mit dem Installationsvorgang normal fort. Anschließend können Sie den Ordner der alten Version löschen und den neuen gegebenenfalls umbenennen.
Anmerkungen zu verschiedenen Versionen ein Update betreffend:

  • Version 0.2.1 (Alpha 10):
    Alle Dateien mit der Endung ".language" im Ordner "languages/de" sollten gelöscht werden. Sie wurden durch entsprechende Dateien mit der Endung ".ini" ersetzt.
  • Version 0.2.0.8.20050121 (Alpha 8) oder älter:
    Der Ordner "language" sollte gelöscht werden, da er nicht mehr benötigt wird. Er wurde durch den Ordner "languages" ersetzt, der auch einen anderen Ordner beinhaltet.

Update vom Tritanium Bulletin Board 1

Seit dem Tritanium Bulletin Board 2 Version Alpha 10 ist ein Konvertierungsscript im Installationsscript integriert. Folgenden Sie einfach den normalen Installationsschritten, bis Sie gefragt werden, ob Sie Daten einer alten "Tritanium Bulletin Board 1"-Installation übernehmen wollen und wählen Sie an dieser Stelle "Ja".
